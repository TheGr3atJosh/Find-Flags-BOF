# Find Flags BOF

This Beacon Object File (BOF) recursively searches a target system for common Capture The Flag (CTF) files. Once identified, it outputs both the full file path and the contents of the flag. 

It is built specifically for penetration testing and CTF environments (such as OSEP labs) to quickly automate the retrieval of proof files.

## 💻 Compatibility

This tool was developed with and intended for **[Adaptix C2]([https://adaptix.ws/](https://adaptix-framework.gitbook.io/adaptix-framework))**, but it is standard BOF code. It is fully compatible with any Command & Control (C2) framework that supports Beacon Object Files.
